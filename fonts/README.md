This folder contains the font files required for generating videos with correct Arabic typography.
The application will automatically download the following fonts if they are missing:

- **Amiri-Regular.ttf**: For general Arabic text and Quranic verses.
- **NotoKufiArabic-Regular.ttf**: For Kufi style text.
- **NotoNaskhArabic-Regular.ttf**: For Naskh style text.
- **NotoNastaliqUrdu-Regular.ttf**: For Nastaliq style text.
- **Tajawal-Regular.ttf**: For modern UI text.

Please do not delete these files unless you want them to be re-downloaded.
